# hermes-skills

Personal Hermes Agent skills collection by [@qiyuey](https://github.com/qiyuey).

## Install

```bash
hermes skills tap add qiyuey/hermes-skills
```

Then install any skill:

```bash
hermes skills install hermes-post-update
```

## Skills

| Skill | Description |
|-------|-------------|
| [hermes-post-update](./autonomous-ai-agents/hermes-post-update/SKILL.md) | Hermes 更新后处理流程 — 分析 update 输出、处理本地变更冲突、运行 doctor 诊断、获取更新日志 |
